# hoi4GFXaid
Small mock-up executable designed with tkinter to drastically reduce workload for gfx modding in hoi4. Takes input of the name of the .dds file, the file path, the prefix, and automatically creates the necessary code for your goals.gfx and goals_shine.gfx files. Also works for event pictures/decisions/leaders provided you don't use the shine, obv.
Program should be self explanatory to use. Half the stuff has defaults anyways, the other half has notes explaining what not to do. Anyone used to GFX modding should have a pretty good idea how to use it.
Also, this shit was made in an hour on pycharm for my own convenience. Do not expect it to be anything other than functional.
